# Rock Paper Scissor 🎮

An interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Play against the computer
- Real-time score tracking
- Random computer choice logic
- Dynamic UI updates

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Play
1. Click Rock, Paper, or Scissors.
2. The computer will generate a random move.
3. The winner is displayed instantly.
